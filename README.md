# Homebrew Tap for zero8dotdev Tools

A collection of beautiful, user-friendly CLI tools by zero8dotdev.

## Installation

```bash
# Add the tap
brew tap zero8dotdev/tools

# Install any tool
brew install warp
```

## Available Tools

- **warp** - Beautiful CLI for Cloudflare WARP - Control your VPN from the terminal

## Updates

```bash
# Update all tools from this tap
brew upgrade
```
